Can you change inverse pendulum height?

Hello everyone,

I was wondering if it is possible to change the inverse pendulum height for Nao walk generation?

We put a Kinect on Nao's head but now it seems less stable while walking. I remember the post on the old forum (users.aldebaran-robotics.com) where it stated that it was possible and that it helped to achieve better performance, but I cannot remember the command which was used.

Is it still possible on the new versions of Naoqi?

Best and thank you in advance, Frano

Hi,

You can change the height of the inverted pendulum with the following python commands:

z = 0.35 #Inverted pendulum height in meters

m = ALProxy("ALMotion", "IP", 9559)

m.setMotionConfig([["WALK_INVERTED_PENDULUM_HEIGHT", z]])

Please note that 0.35 m is the default value for Nao.

Regards,

Juan José
